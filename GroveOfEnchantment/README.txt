Grove of Enchantment is my first game syled in a top down 2D RPG rormat. Created in C# with Unity for Game Design 1101 KSU. To access the game select builds and select the most recent assignment 4 selection. 

Controls

WASD- Movement
J- Attack
K- Interact/Talk
Esc- Exit Game


NOTE: Read Signs and talk to NPCS for a better experience
